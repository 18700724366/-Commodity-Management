# 蜗牛一阶段项目实训1介绍
这是第一阶段的项目，里面包含了对商品的增删改查和对商品进行的购买操作实现了两张表的链接

#### 软件架构
其中有两个app可以用来控制两个程序的互相操作


#### 安装教程

1.  基于eclipse编译
2.  文件格式为GBK
3.  该项目复制就可以正常运行

#### 使用说明

1.  项目部署好了之后就可以直接通过app类编译运行
2.  不需要配置环境
3.  jdk1.8及以上
